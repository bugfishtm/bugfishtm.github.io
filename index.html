Mail Relaying over DNS

License: GPLv3
Description: This web software converts your server into a mail backup relay! It functions independently, allowing users to configure domains for relaying to master mail servers. Additionally, it serves as a panel on a secondary DNS server, fetching registered DNS domains. In essence, if your primary mail server handles mail sending and receiving, this server remains unaffected. However, in the event of the primary server going offline, this secondary server stores incoming mail until the primary server is operational again. The stored mail is subsequently forwarded to the primary server. Users can configure the Master-Mail Server per domain and establish various Relay Servers connected to Mail Domains.
