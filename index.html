    Documentations

    Here you can find documentation about our infrastructure and project documentations! We have a page to quickly check our service and server status here: https://stats.uptimerobot.com/yoLgAPzdsH

    Websites

    This websites are hosted in our infrastructure and serve different purposes.

    bugfish.eu

    This is my main website to showcase all kind of projects and much more: https://bugfish.eu

    cms.bugfish.eu

    Administrator Backend Module of bugfishCMS for us to control our services or for users to register if required: https://cms.bugfish.eu. This website act as our control point.

    forum.bugfish.eu

    This domain will forward you to our Forum: https://forum.bugfish.eu

    lab.bugfish.eu

    Just an internal testing domain to test different bugfishCMS Websites before release: https://lab.bugfish.eu

    petition.bugfish.eu

    A website to promote a petition in germany against corruption: https://petition.bugfish.eu. You can also find the petition here, if you are from germany feel free to support our cause: https://www.openpetition.de/bugfish

    software.bugfish.eu

    This page serves to distribute software. You can find our software store on the following url: https://software.bugfish.eu

    store.bugfish.eu

    This page serves to distribute modules and core updates to bugfishCMS instances. You can find the module and bugfishCMS store on this url: https://store.bugfish.eu

    Services

    If you have access to our services, you can find areas here to login. Besides that you can find information for some services we are hosting.

    Hosting

    If you are the administrator of a hosting account on our primary webmin instance you can login here: https://bugfish.eu:10000

    Webmail

    We have "Usermin" installed for our webserver users to configure theire webmail and more here: https://bugfish.eu:20000/

    DNS

    If you want to use our DNS Servers for hosting Domain Records set up this DNS Servers at your Domain Hoster.

    Primary: thunder.hv-inf.de
    Secondary: shadow.hv-inf.de

    Mail

    Here you can see recommended E-Mail Settings for your E-Mail Client to connect to our server.

    IMAP Hostname: mail.bugfish.eu
    IMAP Username: Your Full E-Mail Address
    IMAP Password: Your E-Mail Password
    IMAP Port: 993
    SMTP Method: SSL
    IMAP Auth: Normal Password

    SMTP Hostname: mail.bugfish.eu
    SMTP Username: Your Full E-Mail Address
    SMTP Password: Your E-Mail Password
    SMTP Port: 587
    SMTP Method: STARTLS
    SMTP Auth: Normal Password

    Cloud

    If you have an account for our files cloud you can login here: https://files.bugfish.eu/index.php/login

    VPN

    We have a SoftEther VPN Server Instance running on our main Server. If you have administrator privilegues you can connect to the default ports or with an registered user on your specific vpn instance.

    Projects

    Access documentation for all my projects. These are included in the respective repositories (docs folder) and work with HTML/JS/CSS, making them accessible on any device without requiring a web server. For any issues or questions, contact us at requests@bugfish.eu.

    bugfishCMS

    License: GPLv3
    A multi-site PHP CMS for quick website deployment. It simplifies setup, customization, and maintenance, ideal for managing multiple sites on a single platform.

    Download Documentation GitHub

    Bugfish Framework

    License: GPLv3
    A robust framework integrating PHP, CSS, and JavaScript to streamline web application development.

    Download Documentation GitHub

    Bind9 Web Manager

    License: GPLv3
    A PHP-based DNS management tool with user administration and replication features, compatible with Plesk, ISPConfig, and Virtualmin.

    Download Documentation GitHub

    Dolibarr Framework

    License: GPLv3
    A Dolibarr module that integrates the Bugfish Framework, adding development tools and logging features.

    Download Documentation GitHub

    Mail Relaying over DNS

    License: GPLv3
    Transforms your server into a mail backup relay, storing and forwarding mail when the primary server is offline.

    Download Documentation GitHub

    Online Book Renting

    License: GPLv3
    A web-based platform for managing book rentals, featuring user management, multi-language support, and ISBN handling.

    Download Documentation GitHub

    Android Web App Example

    License: GPLv3
    An Android Studio project to create a simple web app with navigation controls, easily customizable for your website.

    Download Documentation GitHub

    Dovecot Certificate Interface

    License: GPLv3
    Automates Dovecot per-domain SSL certificate setup, managing ISPConfig configurations or working independently.

    Download Documentation GitHub

    Image Download Manager

    License: GPLv3
    Downloads images from a website, with options to explore subpages and filter high-definition images.

    Download Documentation GitHub

    jQuery Sortselect

    License: GPLv3
    A jQuery plugin that enhances <select> elements with dual sortable select boxes for better UI control.

    Download Documentation GitHub

    Proxmox VMA2RAW

    License: GPLv3
    Converts Proxmox VMA backup files to raw disk images using a Docker image.

    Download Documentation GitHub

    Amnesia Cavelight

    License: GPLv3
    A custom story for Amnesia: The Bunker, crafted for the Frictional Games Modding Jam.

    Download Documentation GitHub
