Bugfish Framework

This is my framework, for faster development and deployment! It contains Javascript/PHP Functions, as PHP Classes with a wide varity of use. Besides that there is a CSS Classes Framework included to make faster style changes on your projects.

Download Documentation Github

Mail Relaying over DNS

Setup a Dedicated Mail Backup MX Server to relay mails in case your main server fails! This will make your incoming mail more secure to receive, because the second server will handle the mail so long as the first server is offline. If the first server comes up again, second MX Server running this software will deliver before timeout mails to the mail MX. Do not use this software inside a running mail environment if you don't know what you are doing. Its best to use this to a mailserver which is dedicated to this purpose. This software can handle Synchronisation with local DNS Zones.

Download Documentation Github

Bind9 Web Manager

DNS Server Replication Manager to work together with bind9! This is one of my flagships. You can manage DNS Replications between bind servers without the need to change bind configuration settings. This script will almost do everything for you. It does not replace a Domain Zone Manager, but it can handle and setup all what is needed to replicat. For more informations check the documentation.

Download Documentation Github

Online Book Renting

Online Book Renting software to publish books and let users rent them! This is a smaller project i made for free for an organisation. Its to manage books, rentings of them and users. Besides that there is a public page which can display enteres books. Quantity and Barcode Management is possible. For more informations see the documentation.

Download Documentation Github

Dovecot Certificate Interface

Manage Dovecot per Domain Mail SSL Certificates with this web software! The reason i coded this software was to control per Domain SSL Certificates together with ISPConfig. This software can run standalone, and is able to sync domains from ispconfig if running together with ispconfig. Do not use this panel together with CMS Systems which self-manage per Domain SSL Certificates (as plesk or virtualmin). For more informations see the documentation.

Download Documentation Github

Bugfish Image Downloader

This is an open source website image downloader for windows OS! I made this in the past to download "images" from specific websites. (Don't think bad about me xDD) It does not serve modern functions like other download managers. But it can be used as a quick solution to fetch images from websites. It supports sub-link image searching and more. For more check the documentation!

Download Documentation Github

Bugfish jQuery Sortselect

This is an jquery plugin for multiple select boxes with sorting function! For sorting functionality you need jquery-ui.min.js! You can easily make cool selectboxes, which support sorting. This is working with "default" html forms and javascript forms as the data is saved in an array and a hidden form element. For more and a demonstration check the documentation page below.

Download Documentation Github
