2. General Information

This is a purely static, non-commercial website. It does not use cookies, does not use analytics or tracking tools, does not embed third-party content (all fonts, images, scripts and audio files are served locally from this website itself), and does not contain contact forms, comment functions or any other input fields that transmit data to us.

Personal data is therefore only processed to the technically necessary extent described below.

3. Hosting via GitHub Pages

This website is hosted on GitHub Pages, a service of GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA ("GitHub").

When you visit this website, GitHub automatically processes technical data that your browser transmits, in particular:

  • IP address of the requesting device
  • Date and time of the request
  • Requested page / file (URL)
  • Browser type and version, operating system (user agent)
  • Referrer URL

This data is processed by GitHub in server log files for the purpose of delivering the website, ensuring its stability and security, and defending against attacks. We ourselves have no access to these log files.

The legal basis for this processing is Art. 6 (1) lit. f GDPR. Our legitimate interest lies in the secure and reliable provision of this website.

GitHub may transfer this data to servers in the United States. GitHub, Inc. is certified under the EU-U.S. Data Privacy Framework, which the European Commission recognises as providing an adequate level of data protection (Art. 45 GDPR).

Further information can be found in GitHub's privacy documentation: GitHub General Privacy Statement and About GitHub Pages — data collection.

4. No Cookies, No Tracking

This website does not set any cookies and does not use web analytics, advertising networks, social media plugins, or fingerprinting technologies. The optional music player and the interactive terminal on the start page run entirely in your browser; no data is transmitted to us or to third parties when you use them.

5. Contact by E-mail

If you contact us by e-mail, the data you provide (your e-mail address, the content of your message and any personal data contained in it) will be processed solely to handle your request. The legal basis is Art. 6 (1) lit. f GDPR (legitimate interest in answering your enquiry) or, if your enquiry is aimed at concluding a contract, Art. 6 (1) lit. b GDPR. Your data will be deleted once the correspondence is complete, unless statutory retention obligations apply.

When you follow an external link, the privacy policy of the respective operator applies. No data is transferred to these providers by merely visiting this website — a connection is only established after you actively click a link.
